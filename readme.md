## Installation

Open your terminal and type in

```sh
$ git clone https://github.com/sitowebveloce/javascript-unsplash-photo-search.git
$ cd java...
open index.html
```

* Clone the repo.
* Add your unsplash key in this file \js\unk.js
* Open index.html file.
